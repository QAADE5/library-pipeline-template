# Fabric Notebooks

- `01_bronze_to_silver.ipynb` and `02_silver_to_gold.ipynb` are pre-built for the Day 4 pipeline lab - import both into a fresh Fabric workspace, edit Cell 1 of the first notebook with your own GitHub org/repo, then wire them into a pipeline.
- Any further notebooks you export from Microsoft Fabric yourself can also be stored here.

## Naming Convention

- `01_bronze_to_silver.ipynb` - Bronze to silver transformation
- `02_silver_to_gold.ipynb` - Silver to gold aggregations
- `03_analysis.ipynb` - Analysis and visualization
